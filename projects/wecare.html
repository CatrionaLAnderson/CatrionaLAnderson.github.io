WeCare Well-Being App

Overview

For my Software Engineering module, we worked with FDMgroup to develop a well-being app that supports their employees. Leading this group project gave me the chance to put my leadership and multi-tasking skills to the test while working with a real-world client. It made me way more confident in guiding a project from start to finish and helped me get better at thinking from a userâ€™s perspective. Plus, I learned how to explain technical stuff in a way that non-tech people could easily understand, which made working with the FDM rep so much smoother.

Key Features

  • User Authentication: Secure login & registration system
  • Profile Management: Allows users and admins to edit personal details
  • Notification System: Sends reminders and updates to users
  • Mental Health Support: Provides resources and assistance for mental health support
  • Fitness Tracking: Allows users to track their fitness progress
  • Admin Panel: Enables admins to manage users and content
  • Ambassador Contact: Enables direct communication with FDM ambassadors

My Role

  • Front-End Development: Built the user interface using React Native & React Native Paper
  • Diet and Fitness tracking: Implemented the diet and fitness tracking feature
  • Ensured consistency of entire app: Maintained consistent design and functionality throughout the application
  • Managed group members: Coordinated tasks and managed the group members effectively
  • UX/UI Design: Focused on creating an intuitive and user-friendly experience

Tools & Technologies Used

  • Front-End: JavaScript, React Native, Expo Go
  • Backend: Node.js, Supabase
  • Design: Figma
  • Methodology: Agile
