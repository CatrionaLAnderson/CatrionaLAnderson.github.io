Aware Phone Interface - Ongoing

Overview

Aware Phone Interface is my Final Year Project designed to enhance user safety and experience when under the influence of alcohol. This project implements innovative interface called "drunk mode", similar to "Do Not Distrub", but this is activated when the user intoxicated. It allows users to set up their prefrences to control their experince and safety when using their phone while drunk. The interface is not only designed to prevent errors but also to reduce cognitive offload for the user based of research and user testing.

Key Features

  • Drunk Mode:Activated by the user - for now at least
  • Typos and Text Correction: Algorithm detects and suggests corrections for drunken texting.
  • UI efficiencyThe UI of the "home screen" is optimised (bigger apps, easy to access emergency buttons etc) to reduce cognitive load for the intoxicated user.
  • Health recommendations Provides alcohol consumption recommendations based off users BMI, and has a calculator for users BAC percentage
  • MUCH MORE! Above are just some of the many features of this interface

Technologies Used

  • Frontend: React Native, Expo
  • Backend: Node.js, Supabase
  • Language: JavaScript
  • Design: Figma
  • Methodology: Agile Development
