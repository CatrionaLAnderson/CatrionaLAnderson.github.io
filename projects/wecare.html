Overview

WeCare is a wellbeing app designed to support FDM employees in maintaining their mental and physical health. The app provides features such as mental health support, fitness tracking, notifications, and contact with ambassadors, ensuring employees have the resources they need for a balanced work-life experience.

Key Features

  • User Authentication: Secure login & registration system
  • Profile Management: Allows users and admins to edit personal details
  • Notification System: Sends reminders and updates to users
  • Mental Health Support: Provides resources and assistance for mental health support
  • Fitness Tracking: Allows users to track their fitness progress
  • Admin Panel: Enables admins to manage users and content
  • Ambassador Contact: Enables direct communication with FDM ambassadors

My Role

  • Front-End Development: Built the user interface using React Native & React Native Paper
  • Diet and Fitness tracking: Implemented the diet and fitness tracking feature
  • Ensured consistency of entire app: Ensured that the app was consistent in terms of design and functionality
  • Managed group members: Managed the group members and assigned tasks
  • UX/UI Design: Focused on creating an intuitive and user-friendly experience

Tools & Technologies Used

  • Front-End: JavaScript, React Native, Expo go
  • Backend: Node.js, Supabase
  • Figma
  • Agile Methodology
