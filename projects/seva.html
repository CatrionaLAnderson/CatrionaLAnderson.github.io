SEVA Smart Glasses Analysis

Overview

This user experience report analyses SEVA (Speech-Enabled Visual Assistant), a conceptual smart glasses system developed by the company Purview to enhance daily life for visually impaired users. The system supports navigation, document reading, AI interaction, and emergency assistance via multimodal input (voice, gesture) and output (audio, AR display).

Design Principles

The report applies Don Norman’s design principles including:

  • Conceptual Model: Voice/gesture-based commands simplify onboarding.
  • Visibility: Clear audio/haptic feedback confirms interactions.
  • Natural Mapping: Input and system response are intuitively aligned.
  • Feedback: Real-time, rich, and spatially aware feedback reinforces user confidence.

Phenomenology & Flow

The report explores phenomenological breakdown using Heidegger’s “ready-to-hand” and “present-at-hand” concepts. When disruptions occur (e.g. misidentification), SEVA’s potential for flow intervention keeps users engaged. This analysis is supported by Csikszentmihalyi’s concept of “flow” in design.

Distributed Cognition

SEVA’s UX is examined through a Distributed Cognition lens, highlighting how the system offloads cognitive tasks across tools, environments, and social actors. Analysis includes memory distribution (internal, external, social), environmental interaction, and multimodal support.
